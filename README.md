## Try it out

### Run in local

```bash
# using pnpm
git clone https://github.com/JoanyBuclon/project-bar.git
pnpm i
pnpm dev


# using npm
git clone https://github.com/JoanyBuclon/project-bar.git
npm i
npm run dev
```
